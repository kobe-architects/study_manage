# 受験ナビ — 学習管理システム

大学受験向けの学習管理システム。`docs/design` のデザイン（学習管理システム.dc.html / 英単語クイズ要件・詳細設計）を実装したもの。

- **フロントエンド**: Vue 3 + TypeScript + Pinia + Vue Router + Vuetify 3（Vite）
- **バックエンド**: PHP 8.3 / Laravel + Sanctum 認証
- **DB**: MySQL 8

## 主な機能

### メインアプリ
- **トップページ** — 全体／科目別の進捗（講義・問題集）、学習ヒートマップ、月間カレンダー（模試予定の登録）、受験カウントダウン、目標設定状況
- **学習項目データ** — 一覧（検索・絞り込み・ページング・追加/編集/削除）、進捗対象ツリー（集計に含める項目をチェックで選択）
- **学習記録** — 科目→大→中→小の連動選択で学習日を登録、今週/連続/累計の統計、ヒートマップ、最近の記録
- **目標設定** — 期限・範囲・進める項目数を設定、達成項目数を学習記録から自動集計
- **システム設定** — アカウント、表示（配色/ナビ/進捗インジケータ）、通知、データ管理

### 英単語クイズ サブシステム
- **クイズ** — 4択 / 入力、SM-2 間隔反復による出題配分、回答の即時採点と例文・習熟度・音声読み上げ、結果表示・誤答リトライ
- **フラッシュカード** — 自動めくり、語/意味の表示時間設定、周回表示
- **復習** — 期間指定で誤答単語を抽出してクイズ／小テスト印刷
- **小テスト印刷** — 意味/スペル/例文穴埋め × 記述/選択、問題用紙・回答用紙・解答シート出力
- **単語帳管理** — CRUD、画像添付、CSV インポート/エクスポート/テンプレート
- **進捗統計** — 総単語数 / 習得 / 学習中 / 新規 / 正答率 / 復習期限到来
- **活動ウォッチ連携** — クイズ/FC 開始時に学習時間計測の確認

## セットアップ

### 前提
- PHP 8.3、Composer、Node.js、MySQL 8（DB `study_manage` 作成済み・root/password）

### バックエンド
```bash
cd backend
composer install
# .env は設定済み（DB: study_manage / root / password、APP_URL: http://localhost:8090）
php artisan migrate:fresh --seed   # スキーマ作成＋デモデータ投入
php artisan storage:link           # 画像配信用（初回のみ）
php artisan serve --host=127.0.0.1 --port=8090
```

### フロントエンド
```bash
cd frontend
npm install
npm run dev      # http://127.0.0.1:5180 （/api は 8090 へプロキシ）
# 本番ビルド: npm run build
```

### ログイン
- メール: `user@example.com`
- パスワード: `password`

## 構成メモ
- 認証はトークンベース（Sanctum）。フロントは `localStorage` にトークンを保持し `Bearer` で送信。
- フロントの `/api` リクエストは Vite プロキシ経由でバックエンド（8090）へ。
- デモデータは `backend/database/data/*.json`（`docs/design` の studyData.js / vocabularyData.js を変換）から、デザインと同一の決定論的アルゴリズムで生成。
- ポートはローカルの他プロジェクトとの競合を避けるため backend=8090 / frontend=5180 を使用。
