# 操作説明書 — 英単語クイズ（UI / 画面操作編）

**KGI/KPI管理ツール / 学習支援機能**

- 文書バージョン: 1.0
- 作成日: 2026-06-29
- 対象: 画面上でユーザーが操作できる機能のみを説明します。内部処理・API・DB等は対象外です。
- 関連文書: 要件定義書_英単語クイズ.md / 詳細設計書_英単語クイズ.md

## 1. はじめに

この説明書は、英単語の学習で使える4つの画面の使い方をまとめたものです。

| 画面 | できること |
|------|------------|
| 単語管理 | 単語の登録・編集・削除、画像添付、検索、Excel取込/出力 |
| クイズ | 4択 / 入力形式で出題・採点。小テストの印刷もここから |
| フラッシュカード | 「語→意味」を自動でめくって高速インプット |
| 間違えた単語の復習 | 過去に間違えた単語だけを再出題 |

### 画面への入り方

「教材・講義」の詳細画面 →「単語帳」セクションから、以下のボタンで各画面に移動します。

```
[ 単語管理 ]  [ クイズ ]  [ フラッシュカード ]  [ 間違えた単語の復習 ]
```

各画面の左上にある「←（戻る）」ボタンで前の画面に戻れます。

## 2. 単語管理画面

単語の一覧表示・追加・編集・削除を行う画面です。

### 2.1 単語一覧（表）

表には次の列が表示されます。

| 列 | 内容 |
|----|------|
| # | 並び順 |
| 単語 | 英単語（画像があれば小さなアイコンも表示） |
| 意味 | 日本語の意味 |
| 品詞 | 名/動/形 など |
| 重要度 | −（無印） / * / ** |
| ラベル | 易（緑） / 普 / 難（赤）の色付きチップ |
| 習熟度 | 高 / 中 / 低 の色付きチップ（未設定は −） |
| 学習 | 「正解数/回答数」の比率チップ（正答率で色が変化） |
| 操作 | ✏️編集 / 🗑️削除 ボタン |

- 行をクリックして展開すると、**例文・和訳・例文説明**が表示されます（未登録の場合は「例文情報なし」）。

### 2.2 検索・絞り込み

表の上部で、次の条件で表示を絞り込めます。

- **検索ボックス**: 単語・意味・例文・和訳の文字列で検索
- **セクション**: 特定のセクションだけ表示
- **ラベル**: 易 / 普 / 難 で絞り込み
- **重要度**: 無印 / * / ** で絞り込み
- **習熟度**: 高 / 中 / 低 で絞り込み

### 2.3 単語の追加・編集（入力ダイアログ）

「追加」ボタン、または各行の ✏️編集 ボタンで入力ダイアログが開きます。

| 入力項目 | 必須 | 補足 |
|----------|------|------|
| 単語 | ○ | 英単語 |
| 意味 | ○ | 日本語の意味 |
| 品詞 | − | 「名 / 動 / 形 / 副 など」 |
| 重要度 | − | 0 / 1 / 2（初期値 1） |
| ラベル | − | 易 / 普 / 難（初期値 普） |
| 習熟度 | − | 未設定 / 高 / 中 / 低 |
| メモ | − | 自由記述 |
| 例文 | − | 英語の例文 |
| 和訳 | − | 例文の日本語訳 |
| 例文説明 | − | 例文の補足説明 |
| 画像 | − | JPEG/PNG/GIF/WebP。プレビュー表示、差し替え・削除可 |

- 「単語」「意味」が未入力だと保存できません。
- 新規追加時は、前回入力した内容が途中まで自動で復元されます。
- 下部の「キャンセル」「保存」で操作を確定します。

### 2.4 削除

- 各行の 🗑️ボタンで個別削除します。
- 単語を削除すると、その単語に添付した画像も一緒に削除されます。

### 2.5 Excelでの取り込み・書き出し

- **テンプレートのダウンロード**: 取り込み用のExcelひな形を入手できます。
- **インポート**: Excel/CSVファイルを選んで単語を一括登録します。完了後、取込件数とスキップ件数が表示されます。
- **エクスポート**: 現在の単語帳をExcelファイルとして書き出します。

## 3. クイズ画面

### 3.1 出題条件の設定

クイズ画面を開くと、まず設定画面が表示されます。

| 設定項目 | 操作 | 初期値 |
|----------|------|--------|
| セクション | チップで複数選択（全選択/全解除ボタンあり） | すべて選択 |
| 重要度 | 無印 / * / ** を複数選択 | 指定なし（全件） |
| ラベル | 易 / 普 / 難 を複数選択 | 指定なし（全件） |
| クイズタイプ | 「4択クイズ」/「英単語入力」を選択 | 4択クイズ |
| 問題数 | スライダー（5問単位）/「すべて」切替/プリセット（10・20・50・全部） | すべて |
| 出題順序 | 「ランダム」/「順番通り」 | ランダム |

- セクションを1つも選んでいないと「クイズ開始」は押せません。
- 画面下部のボタン:
  - **クイズ開始** … 設定した条件でクイズを始めます
  - **小テスト印刷** … 紙の小テストを作ります（→ 第6章）

### 3.2 4択クイズの解き方

画面上部に進捗バーと「現在の問題数 / 全問題数」が表示されます。

1. **英単語が表示されます**（単語をタップすると発音が再生されます）。
   - 品詞・重要度（★）・ラベル・習熟度も一緒に表示されます。
2. 「**選択肢を表示**」ボタンで A/B/C/D の4つの意味が出ます。
3. いずれかを選ぶと採点され、正解＝緑、不正解＝赤で表示されます。
   - 答えがわからないときは「**わからない**」ボタンを押せます（不正解扱い）。
4. 回答後は次の情報が表示されます。
   - 正解 / 不正解のメッセージ
   - 例文（対象の単語が太字で強調、タップで発音）
   - 「和訳を表示」トグル、例文説明
   - 単語の画像（あれば）
5. **習熟度の更新**: 「更新しない / 高 / 中 / 低」のボタンでその場で習熟度を変更できます。
6. **メモの編集**: メモのチップをタップして編集できます。
7. 「**次へ**」で次の問題へ進みます。

### 3.3 英単語入力クイズの解き方

1. **日本語の意味が表示されます**。
2. 入力欄に英単語を入力します（自動でカーソルが入ります）。
3. **Enterキー**または送信ボタンで回答します。
   - 大文字・小文字の違いは無視して採点されます。
4. 回答後は4択と同じく、正誤・例文・習熟度更新・メモ編集・画像が表示されます。
5. 「次へ」で次の問題へ進みます。

### 3.4 前の問題に戻る

- 2問目以降では「**前の問題**」ボタンが表示され、直前の問題に戻れます。
- 回答済みの問題に戻ると、回答結果が復元された状態で表示されます。

### 3.5 結果画面

全問終わると結果画面が表示されます。

- **正答数 / 出題数** と **正答率** を大きく表示（80%以上=緑、50〜79%=橙、50%未満=赤）。
- 間違えた単語の一覧（「答え ON/OFF」で意味の表示を切替）。
- ボタン:
  - **間違えた単語だけリトライ** … 誤答した単語だけで再挑戦
  - **新しいクイズ** … 設定画面に戻って別の条件で出題

## 4. フラッシュカード画面

「語 → 意味」を自動でめくって表示し、流し見で覚えるモードです。

### 4.1 表示条件の設定

| 設定項目 | 操作 | 初期値 |
|----------|------|--------|
| セクション | 複数選択（全選択/全解除） | すべて選択 |
| 重要度 | 無印 / * / ** 複数選択 | 指定なし |
| ラベル | 易 / 普 / 難 複数選択 | 指定なし |
| 単語数 | スライダー / 「すべて」/ プリセット | − |
| 出題順序 | ランダム / 順番通り | ランダム |
| 単語表示時間 | 2〜10秒 | 5秒 |
| 意味表示時間 | 2〜10秒 | 3秒 |

- 「スタート」で開始します。

### 4.2 表示中の操作

- 全画面で、**英単語が表示され発音が流れた後、意味が表示**されます。
- 設定した時間が経つと自動的に次の単語に進みます。
- 最後まで行くと最初に戻り、「○周目」と周回数が表示されます。
- **画面のどこか（ボタン以外）をタップ**すると一時停止／再開します。
  - 再生 / 一時停止ボタン、停止ボタンも下部にあります。
- 別のタブに切り替えると自動で一時停止します。
- 上部に「現在 / 総数（○周目）」と進捗バーが表示されます。
- 停止または ×ボタンで設定画面に戻ります。

## 5. 間違えた単語の復習画面

過去に間違えた単語だけを集めて復習できます。

### 5.1 期間の指定

次のいずれかを選びます。

- 過去3日 / 過去7日 / 過去1ヵ月
- 任意日付選択（開始日・終了日を指定）

クイズタイプ（4択 / 入力）も選んで「**検索**」を押します。

### 5.2 結果と開始

- 該当期間に間違えた単語の件数と一覧が表示されます。
- 間違えた単語があれば、次の操作ができます。
  - **クイズ開始** … その単語だけでクイズを実施
  - **小テスト印刷** … その単語で小テストを作成
- 間違えた単語がなければ「この期間に間違えた単語はありません」と表示されます。

## 6. 小テストの印刷

クイズ画面または復習画面の「**小テスト印刷**」ボタンから利用します。

### 6.1 印刷オプションの選択

| 項目 | 選択肢 |
|------|--------|
| テストタイプ | 意味を問う / スペルを問う / 例文穴埋め |
| 回答方式 | 記述式 / 選択式 |

- 「例文穴埋め」は、例文が登録されている単語だけが対象になります。

### 6.2 印刷プレビューと出力

印刷シートが画面に表示されます。出力する範囲を選べます。

- **すべて** / **問題のみ** / **回答用紙のみ** / **解答のみ**
  - ※「例文穴埋め」では回答用紙のみは選べません。

それぞれ次の内容が含まれます。

- **問題用紙**: 番号付きの問題（選択式の場合は A/B/C/D の選択肢付き）
- **回答用紙**: 日付・氏名欄と解答欄（選択式はマーク欄、記述式は解答線）
- **解答**: 各問題の正解（品詞やメモも表示）

「印刷」ボタンを押すとブラウザの印刷画面が開きます。閉じるボタンで元の画面に戻ります。

## 7. 便利機能・ヒント

| 機能 | 説明 |
|------|------|
| 発音再生 | クイズ・フラッシュカードで英単語や例文をタップすると音声が流れます（女性の声を優先）。 |
| 習熟度のその場更新 | クイズの回答後に「高/中/低」をワンタップで設定でき、苦手管理に役立ちます。 |
| メモのその場編集 | 回答画面からメモを追記できます。 |
| 自動の優先出題 | 通常のクイズでは、復習すべき単語・新しい単語・苦手な単語がバランスよく出題されます。 |

## 8. よくある操作の流れ

**毎日の暗記**
1. クイズ画面 → 条件はそのまま「クイズ開始」
2. 4択で解く → 間違えたら習熟度を「低」に
3. 結果画面で「間違えた単語だけリトライ」

**試験前の総復習**
1. フラッシュカードで全体を流し見
2. 復習画面で「過去7日」の誤答を再クイズ
3. クイズ画面の「小テスト印刷」で紙の確認テストを作成

**新しい単語帳の準備**
1. 単語管理 →「テンプレート」を入手
2. Excelに入力 →「インポート」で一括登録
3. 必要に応じて画像や例文を追加
